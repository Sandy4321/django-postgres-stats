Installation instructions
=========================

Download and installation
-------------------------

Installation with pip
^^^^^^^^^^^^^^^^^^^^^

You can use pip to install django-postgres-power::

  $ pip install django-postgres-power


Using in Django
^^^^^^^^^^^^^^^

You will need to add the *postgres_power* application to the INSTALLED_APPS
setting of your Django project *settings.py* file.::

    INSTALLED_APPS = (
        ...
        'postgres_power',
    )
